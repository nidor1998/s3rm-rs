# Suggested Commands

## Build & Run
- `cargo build` - Build the project
- `cargo build --release` - Build release version
- `./target/debug/s3rm-rs --help` - Run CLI

## Testing
- `cargo test` - Run all tests
- `cargo test -- --nocapture` - Run tests with output
- `cargo test --test <test_name>` - Run specific test file

## Code Quality
- `cargo fmt` - Format code (CI enforces this)
- `cargo clippy` - Run linter
- `cargo check` - Fast syntax validation
- `cargo deny check` - Check dependencies for vulnerabilities

## System Utils
- `git`, `ls`, `cd`, `grep`, `find` - Standard Linux commands
